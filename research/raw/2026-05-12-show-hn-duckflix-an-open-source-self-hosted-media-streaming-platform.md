## Extracted article text

High-performance media streaming backend built with Bun and TypeScript.
- Runtime: Bun
- Framework: Elysia.js
- Database: SQLite with Drizzle ORM
- Media Processing: FFmpeg
- Torrent Client: ikatson/rqbit
- Validation: Zod
- Authentication: JsonWebToken
- Email: Nodemailer
- Logging: Pino
- Metadata: TMDB API
- Subtitles: OpenSubtitles.com
See BUILDING.md for full setup instructions.
