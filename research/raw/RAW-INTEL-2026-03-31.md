## What changed
Ollama has introduced several new features and improvements, including compatibility with Anthropic API for Claude Code, image generation capabilities, and enhanced performance on Apple Silicon devices using MLX. Additionally, Ollama now offers a simplified setup process through the `ollama launch` command.

## Why it matters
These advancements enhance AI productivity by providing more flexible and efficient tools for developers and businesses. The compatibility with Anthropic API allows for greater control over AI implementations, while image generation capabilities open up new creative possibilities. Enhanced performance on Apple Silicon devices ensures better user experience and faster processing times.

## Source-backed notes
- Ollama's compatibility with Claude Code through the Anthropic Messages API enables developers to use agentic coding tools locally or via cloud models (https://ollama.com/blog/claude).
- Image generation capabilities are now available on macOS, with support for Windows and Linux coming soon (https://ollama.com/blog/image-generation).
- Ollama's performance has been significantly improved on Apple Silicon devices using MLX, a machine learning framework from Apple (https://ollama.com/blog/mlx).
