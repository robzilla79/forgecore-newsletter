Here is a summary of the news articles from the Ollama Blog:

1. Image generation (experimental): Ollama now supports image generation on macOS, with Windows and Linux support coming soon. The 6 billion parameter text-to-image model Z-Image Turbo generates photorealistic images and handles bilingual text rendering in both English and Chinese.
2. MiniMax M2: MiniMax M2 is a new model available on Ollama's cloud service, built for coding and agentic workflows. It demonstrates highly competitive general intelligence across various tasks and excels at coding-related tasks.
3. New coding models & integrations: GLM-4.6 and Qwen3-coder-480B are available on Ollama's cloud service with easy integrations to popular tools like VS Code. Qwen3-Coder-30B has been updated for faster, more reliable tool calling in Ollama's new engine.
4. NVIDIA DGX Spark performance: Performance tests were run on the latest NVIDIA DGX Spark firmware and an updated Ollama version to see how Ollama performs. The tests showed that Ollama delivers lower latency, lower cost, and higher throughput for interactive agents and batched sampling.

These updates demonstrate Ollama's commitment to providing advanced tools for various use cases, including image generation, coding, and agentic workflows.
