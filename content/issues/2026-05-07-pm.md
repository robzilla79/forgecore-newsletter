### Example Code Snippet
If you're using Repurpose.io's API, you can automate the posting process with a simple script:

```python
import requests

def post_content(api_key, content):
    url = "https://api.repurpose.io/v1/post"
    headers = {"Authorization": f"Bearer {api_key}"}
    response = requests.post(url, headers=headers, json=content)
    return response.json()

# Example content to post
content = {
    "title": "New Podcast Episode",
    "description": "Check out our latest episode on AI tools for solo founders!",
    "platforms": ["YouTube", "Facebook"]
}

# Post content
api_key = "your_api_key_here"
result = post_content(api_key, content)
print(result)
```
