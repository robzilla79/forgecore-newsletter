## Hook
Ollama redefines local AI capabilities with groundbreaking updates: **MLX-powered performance on Apple Silicon**, seamless **image generation**, and simplified workflows via `ollama launch`. ## Top Story
Ollama's latest updates focus on performance optimization, new features, and integration with AI tools. Key developments include:

1. **Performance Boost**: Ollama now leverages Apple's MLX framework for Apple Silicon devices (M5, M5 Pro, M5 Max), enabling faster execution of AI workloads.
2. **GPU Neural Accelerators**: Enhanced TTFT and generation speed on Apple's latest chips.
3. **Simplified Setup**: Launches coding tools like Claude Code, OpenCode, and Codex with local or cloud models.
4. **No Configuration Needed**: Eliminates the need for environment variables or config files.
5. **Compatibility**: Codex CLI can use open models like `gpt-oss:20b` or `gpt-oss:120b`.
6. **Cloud Models**: All Ollama Cloud models work with Codex (e.g., `gpt-oss:120b-cloud`).
7. **Functionality**: Connects messaging tools and automates tasks.
