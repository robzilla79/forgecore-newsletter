# ForgeCore Publishing SOP

## Site purpose
ForgeCore is an AI tools newsletter and website for builders, creators, and solo operators.
The business goal is to generate:
- SEO traffic
- Newsletter subscribers
- Affiliate revenue
- Display ad revenue over time

## Audience
ForgeCore serves:
- Solo founders
- Creators
- Builders
- Indie hackers
- Operators
- Small business owners

## Content promise
Every article must help the reader:
- Save time
- Make money
- Automate work
- Build a system
- Choose an AI tool
- Avoid wasting money on the wrong tool

## Publishing workflow
1. Pick one high-intent AI tools topic.
2. Write the website article.
3. Create a newsletter version.
4. Add SEO metadata.
5. Add affiliate placeholders.
6. Add internal link suggestions.
7. Save the article in the correct site format.
8. Commit changes to GitHub.
9. Cloudflare deploys from the main branch.
